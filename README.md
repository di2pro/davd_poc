To run project locally use docker compose with `.local.env` file
```.env
SETTINGS=base
DJANGO_SECRET_KEY=...
DJANGO_ALLOWED_HOSTS=...
POSTGRES_DB=...
POSTGRES_USER=...
POSTGRES_PASSWORD=...
DB_HOST=postgres
DB_PORT=5432
```
